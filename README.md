# Breath of Earth - Air Quality Game

An interactive web application that educates users about air quality through an immersive game experience. Users can explore cities, view real-time air quality data, and play a pollution-fighting game.

## Features

- **Interactive Globe**: Explore cities worldwide with real-time air quality data
- **City Visualization**: Detailed air quality information with historical trends
- **Pollution Game**: Clean air mini-game where users fight different pollutants
- **Educational Content**: Learn about air pollution sources and health impacts
- **Responsive Design**: Works on desktop and mobile devices

## Technologies Used

- React 18 with TypeScript
- Vite for fast development
- Tailwind CSS for styling  
- React Router for navigation
- Lucide React for icons
- Vanta.js for 3D background effects

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Open your browser to `http://localhost:3000`

## Game Instructions

1. Enter a city name on the landing page
2. Explore the city's air quality data and historical trends
3. Click "Play Clean Air Game" to start the pollution-fighting mini-game
4. Select appropriate solutions (trees, wind energy, solar panels) to clear pollutants
5. View your score and learn about air quality improvement strategies

## Building for Production

```bash
npm run build
```

The built files will be in the `dist` directory.
